Code with Mashwani — SQL Mastery : Beginner to Job Ready
A self contained, web app that teaches SQL from the very first SELECT statement all the way to interview-ready mastery. Built as a 30 day structured curriculum with daily lessons, hands on practice, real interview questions, and 400 game style challenges.

Overview
Code with Mashwani is a personal SQL learning platform designed for anyone preparing for data analyst, data scientist, or business analyst roles. Instead of scattered tutorials and disconnected practice sites, everything lives in one place: concept explanations, hands on drills, a searchable cheatsheet, and real questions asked at top tech companies.

Features
30 Day Lesson Path
A day by day curriculum that starts with SELECT and FROM and builds up through filtering, sorting, aggregation, and grouping. Each day introduces one new concept in plain English before giving you challenges to solve, so you're never thrown into practice without context.

Joins Lab
Every join type : INNER, LEFT, RIGHT, FULL OUTER, CROSS, SELF, and UNION / UNION ALL — gets its own dedicated concept page with real tables shown side by side, a simple English explanation of what's actually happening, guidance on when to use it, and 10 practice challenges per join, with the last 5 framed as real workplace scenarios.

Theory & Viva
44 essential concepts covering SQL fundamentals, keys and design, joins and sets, filtering and aggregation, window functions, CTEs and subqueries, and transactions including a dedicated section for Data Analyst and Data Scientist specific interview questions. Each concept has a short answer up front plus a full explanation and example query.

Interview Prep
Real problems asked at companies like Amazon, Google, and Meta, tagged by topic and difficulty. Each problem includes a key idea, a step by step approach broken into numbered stages, and a full solution you can reveal once you've thought it through yourself.

SQL Games
20 games, 20 levels each, for 400 total challenges — all framed as real-world scenarios instead of dry exercises. Solve crimes as a SQL Detective, run a restaurant's database, investigate a bank heist, manage a hospital's records, analyze a music streaming chart, run an airline's operations, and more.

Cheatsheet
A searchable, categorized quick reference covering basic commands, WHERE filters, sorting, aggregate functions, joins, window functions, CTEs, subqueries, CASE statements, date functions, string functions, NULL handling, set operations, and interview tips and built for quick lookup while you're actually writing queries.

Progress Tracking
XP and streak tracking, per lesson and per game completion percentages, and an overall progress bar so you always know where you left off.

Account System
Simple email and password login with account creation, and progress saved locally in the browser — no server required.

Tech Stack
Single-file HTML application
Inter and IBM Plex Mono via Google Fonts
No backend, no database : runs entirely client side
Progress persisted in local browser storage
Getting Started
Clone this repository
Open index.html in your browser
Create an account and start with Day 1
No build step, no dependencies to install, no server to run.

Who This Is For
Anyone learning SQL for a data role from someone writing their first SELECT statement to someone brushing up on window functions and CTEs before a technical interview.
