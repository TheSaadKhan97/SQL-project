# Code with Mashwani — SQL Mastery: Beginner to Job Ready

A comprehensive SQL learning platform designed to take learners from their first `SELECT` statement to interview ready SQL proficiency. Built as a single file web application, the platform combines structured lessons, hands on practice, interview preparation, gamified challenges, and a searchable SQL reference into one learning experience.

---

## Overview

**Code with Mashwani — SQL Mastery: Beginner to Job Ready** was created to provide a structured alternative to scattered SQL tutorials and disconnected practice websites.

The application guides learners through a carefully designed learning path that begins with SQL fundamentals and progresses to advanced concepts commonly tested in technical interviews for Data Analyst, Business Analyst, and Data Scientist roles.

Everything runs entirely in the browser without requiring a backend server or database.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/TheSaadKhan97/SQL-project.git
```

Navigate to the project folder:

```bash
cd SQL-project
```

Open the application:

```text
Code With Mashwani.html
```

Create an account and begin learning from **Day 1**.

No installation, dependencies, or server setup is required.
